CSS- Stands for "Cascading Style Sheet." Cascading style sheets are used to format the layout of Web pages. They can be used to define text styles, table sizes, and other aspects of Web pages that previously could only be defined in a page's HTML.

ECMAScript - stands for "European Computer Manufacturers Association Script". It a scripting language based on JavaScript. ECMAScript is a standard script language, developed with the cooperation of Netscape and Microsoft. The widely-used scripting language is used in Web pages to affect how they look or behave for the user.
